# ServiBoard - Service Management System

## Description
This system was developed as part of academic coursework focusing on service management and financial workflow automation.

A web-based service management system developed using PHP and MySQL.

## Features
- Request management system
- Financial module (collections, disbursement, reporting)
- User-based access control
- Workflow and approval system

## Modules
- Financial Management
- HR Requests
- Logistics Requests
- User Management

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript

## Setup Instructions
1. Import the database.sql file into MySQL
2. Configure database connection in /config folder
3. Run the project using XAMPP or similar server
